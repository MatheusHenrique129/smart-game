# Smart Games

Front-End do projeto para a loja de jogos, da empresa Smart Games

### Linguagem utilizada: JavaScript

### Framework: ReactJS

### Bibliotecas de design:

- styled-components

### Como instalar no seu computador

$ git clone https://github.com/MatheusHenrique129/smart-game.git

### Acesse a pasta em que você rodou o git clone

$ cd "nome_pasta"

### Instalar dependências do projeto

$ npm install

### Execultar a aplicação

$ npm start
